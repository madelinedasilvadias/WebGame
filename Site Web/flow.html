Sitemap



The game site layout:

Our game will consist of four pages. The first page will be used for registration before entering the game. The second one will be dedicated for the rules of the game. Then, the following one will showcase the game, while the last page will display the results of other players as well as our own. From the results page, players will have the option to replay or return to the registration page.



Home page:

Before starting to play, the player's nickname must be entered. If the user selects a nickname that has been used before, they will be redirected to the home page to choose an unused one.
